# PWA Icons Guide

To create proper icons for your PWA, you'll need the following files in the `public` folder:

## Required Icon Files

1. **icon-192.png** (192x192 pixels)
2. **icon-512.png** (512x512 pixels)
3. **favicon.png** (32x32 or 64x64 pixels)

## How to Create Icons

### Option 1: Use Online Tools (Easiest)
1. Go to https://www.pwabuilder.com/imageGenerator
2. Upload your logo (preferably 512x512 or larger)
3. Download the generated icon pack
4. Copy the icons to this `public` folder

### Option 2: Use Design Software
1. Open your design software (Photoshop, Figma, Canva, etc.)
2. Create a 512x512 canvas with the app logo
3. Use these colors for the AyurvedaGPT theme:
   - Background: `#053445` (Deep teal)
   - Logo/Text: `#6DB4CD` (Soft teal)
4. Export as PNG:
   - 512x512 ‚Üí `icon-512.png`
   - 192x192 ‚Üí `icon-192.png`
   - 32x32 ‚Üí `favicon.png`
